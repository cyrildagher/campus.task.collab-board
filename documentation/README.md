# Documentation

This folder contains the complete project documentation for the Campus Task Collaboration Board.

## Files

- **PROJECT_DOCUMENTATION.html** - Complete documentation in HTML format (ready to print to PDF)
- **PROJECT_DOCUMENTATION.md** - Complete documentation in Markdown format
- **images/** - All screenshots and images used in the documentation

## How to Use

### View the HTML Documentation
1. Open `PROJECT_DOCUMENTATION.html` in any web browser
2. All images should load automatically from the `images/` folder

### Convert to PDF
1. Open `PROJECT_DOCUMENTATION.html` in your browser
2. Press `Ctrl+P` (or File â†’ Print)
3. Select "Save as PDF" as the printer
4. Save the PDF file

### Alternative: Use Markdown
- The Markdown file can be converted to PDF using tools like:
  - Pandoc: `pandoc PROJECT_DOCUMENTATION.md -o PROJECT_DOCUMENTATION.pdf`
  - Online Markdown to PDF converters
  - VS Code extensions
